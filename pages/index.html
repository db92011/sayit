How it works

Daily life creates emotionally loaded conversations. You are trying to get something across without hurting someone, but the version in your head can come out too sharp, too messy, or too reactive.

1. Start with what you actually want to say

Type it out or talk it through exactly as it is coming up for you, with context if you need it.

2. Refine it for the relationship

Tell SayIt who it is for and how you want it to land, then let it clean up the delivery without losing the point.

3. Read it, rehearse it, or send it

Use the final draft as a text, email, or conversation guide, and switch to teleprompter mode when you need to say it aloud.

Install

Use the install flow built into your device. This page is the correct place to do it.

iPhone / iPad

  1. Open this page in Safari.
  2. Tap Share.
  3. Choose Add to Home Screen.
  4. Open SayIt from your Home Screen.
If you see the marketing URL in the sheet, back out and return to this app-domain page first.

Android

  1. Open this page in Chrome.
  2. Use the menu and choose Install app or Add to Home screen.
  3. Open SayIt from your home screen.
Chrome will install from this page and then launch into the real app shell.

Mac / Windows

  1. Open this page in Chrome, Edge, or Safari.
  2. Use the browser’s install option or add it to your Dock.
  3. Open SayIt from the saved app icon.
The installed shortcut launches the SayIt app instead of this installer page.
